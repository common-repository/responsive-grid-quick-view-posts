Creating grid quick view posts

== Description ==
You have a WordPress website, you are looking for a module to display posts in a unique way to attract viewers on your site.
Come with "Responsive Grid Quick View Posts for WordPress", a great module to display posts in grid interface, beautiful image effects, users can quick look post content in popup window.
Administrators can set up number of columns to displayed on the grid interface, show or hide the information …
Beautifully, attracting viewers, quickly, convenient, saves time of viewers, they are certainly will more enjoy and stay longer on your site.

== Installation ==
1. Upload \"grid-quick-view\" folder to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Add \"[smartcms_grid_quick_view]\" as shortcode or create a widget.
